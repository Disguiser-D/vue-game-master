## 通过本项目你可以了解到
- Vue.js基本语法

- 利用Vue-cli简单搭建一个项目
- Vue-Router的基本使用
- element ui 的基本使用
- axios访问数据的封装
- JAVA&JDBC
- Vue路由守护和HTTP拦截器
